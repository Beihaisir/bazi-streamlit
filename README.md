# 命理大师问答详批系统 V6

V6 在 V5 的基础上升级为“问答式大师断盘”。

## 核心功能

- 用户输入出生时间和出生地址
- 后台自动定位、自动真太阳时、自动排盘
- 自动计算四柱、十神、五行、身势、大运、流年
- 自动输出大师总断
- 自动输出出生至今十件应事
- 自动输出事业、财运、婚恋、健康建议
- 新增“大师问答”：
  - 用户输入具体问题
  - 系统自动识别问题方向
  - 根据命盘、当前大运、流年、未来十年趋势生成针对性断语
- 命盘依据放在最后，供复核使用

## 运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub 部署

上传到 GitHub 后，用 Streamlit Community Cloud 部署，入口文件：

```text
app.py
```

## 说明

本系统为传统命理与象征性分析工具，不替代法律、医学、投资、职业等专业判断。
