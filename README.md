# 八字趋利避害决策系统 V2

## 功能

- 输入公历出生日期和时间
- 自动生成四柱八字
- 自动计算天干十神、地支藏干十神
- 简化判断日主强弱
- 自动给出初步喜忌建议
- 支持手动校准喜用神、忌神、能力链条
- 支持历史事件校准
- 支持重大生活决策评分
- 输出绿灯、黄灯、红灯
- 导出 JSON 报告和历史事件 CSV

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
streamlit run app.py
```

## 注意

本工具为决策辅助原型，不构成法律、医学、投资、职业建议。
