# 八字命理测算系统 V4

## 定位

V4 不再以“决策评分”为主体，而是以完整八字命理详批为主体。

用户只需输入：

- 姓名/代号
- 性别
- 出生年月日时分
- 出生地址

系统自动完成：

- 地址自动定位
- 真太阳时换算
- 四柱排盘
- 十神计算
- 五行旺衰
- 日主强弱
- 喜用神与风险十神
- 格局/能力链条识别
- 神煞简表
- 大运流年
- 出生至今最具代表性的十件事情
- 事业、财运、婚恋、健康、避坑建议
- 附属决策评分

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
streamlit run app.py
```

## 部署

上传到 GitHub 后，可用 Streamlit Community Cloud 部署。

入口文件：

```text
app.py
```

## 注意

本工具为传统命理与象征性分析系统，不构成法律、医学、投资、职业等专业建议。
