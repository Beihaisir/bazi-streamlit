# 命理大师详批系统 V5

V5 的目标不是做一个“表格型排盘工具”，而是做成更像真人命理师的详批系统。

## 核心变化

- 前台以“大师总断”为主，而不是表格
- 自动排盘、真太阳时、大运流年全部后台完成
- 自动输出出生至今十件最具代表性的应事年份
- 事业、财运、婚恋、健康以断语形式输出
- 底层命盘依据放到最后，供复核使用
- 地址只需输入文字，后台自动定位并换算真太阳时
- 附属决策模块保留，但不再喧宾夺主

## 运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 部署

上传 GitHub 后，用 Streamlit Community Cloud 部署，入口文件填：

```text
app.py
```

## 说明

本系统为传统命理与象征性分析工具，不替代法律、医学、投资、职业等专业判断。
