# 八字趋利避害决策系统 V3

## V3 更新

- 修复日期选择问题：改用年/月/日/时/分数字输入，避免浏览器日期控件不可选的问题
- 支持真太阳时：经度修正 + 均时差近似
- 自动排四柱八字
- 自动计算天干十神、地支藏干十神、藏干权重
- 增加纳音、空亡、十二长生、神煞简表
- 增加五行季节旺衰统计
- 增加简化日主强弱判断
- 增加格局/能力链条提示
- 增加大运顺逆与大运柱序列
- 保留历史校准和生活决策评分

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
streamlit run app.py
```

## 部署

上传到 GitHub 后，在 Streamlit Community Cloud 选择仓库、分支和 `app.py` 部署。

## 注意

本工具为决策辅助原型，不构成法律、医学、投资、职业建议。不同命理流派对排盘、真太阳时、起运和格局判断存在差异。
