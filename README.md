# 高级八字命理报告 V8

V8 在 V7 的完整高级报告结构上，重点升级为“细致人话版”：每个模块都按先结论、再解释、再现实建议、最后风险提醒的方式输出。

## 功能

- 出生地址自动定位
- 真太阳时换算
- 基础八字排盘
- 五行分析图表
- 十神结构分析
- 神煞详解
- 性格分析完整版
- 事业分析完整版
- 财运分析完整版
- 情感分析完整版
- 健康分析完整版
- 大师建议完整版
- 人生大运周期解读
- 未来十年运势图表
- 流年详批：事业、财运、感情、健康、人际五维评分
- 九宫飞星风水布局
- 12个月行动日历
- 出生至今十件代表性应事
- JSON 高级报告导出

## 运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub 部署

上传到 GitHub 后，用 Streamlit Community Cloud 部署，入口文件填：

```text
app.py
```

## 说明

本系统为传统命理与象征性分析工具，不替代法律、医学、投资、职业等专业判断。


## V8 升级重点

- 每一项解读更细，不再只给短句
- 总览、性格、事业、财运、情感、健康全部改为人话详批
- 大运周期增加每步运的现实解释
- 流年五维增加年度总断、强项、弱项和行动原则
- 12个月行动日历增加逐月人话建议
- 保留完整表格和图表，但解读不再依赖用户自己看表
